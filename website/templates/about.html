About


MHR Builds is a website that allows you to build your own Monster Hunter Rise gearsets.

This website is a work in progress and is not yet complete.

Features:

- Login System

- Create Gearsets (Public or Private)

- Add pieces of gear and decorations as available to specified gearset

- View your own gearsets and change as desired (delete or add new items)

- Ability to see skills currently set in gearset (set up similarly to the game's system, see how many available vs. how many you currently have in the gearset

- View Public Gearsets

- Filtered by weapon type

- Delete your own Gearsets

- View skill list and see details about skills

- Upon clicking on skill, a window will be displayed showing all pieces of gear and decorations that provide said skill

- This is also available from the Gearset Details screen

- All tables are searchable using the search bar in navigation. Can search by any field to narrow down possibilities

- Will likely not be adding Low Rank gear, but High and Master rank gear is available
